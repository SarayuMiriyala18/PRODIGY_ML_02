Task-02: Customer Segmentation using K-Means Clustering

Objective

To implement the K-Means Clustering algorithm and group retail store customers based on their annual income and spending score.

Dataset

Customer Segmentation Dataset

Dataset Link:
https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python

Tools and Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

Algorithm Used

K-Means Clustering

Features Used

- Annual Income (k$)
- Spending Score (1-100)

Steps Involved

1. Import required libraries.
2. Load the customer dataset.
3. Select Annual Income and Spending Score as features.
4. Apply K-Means Clustering.
5. Divide customers into different clusters.
6. Visualize customer segments using a scatter plot.
7. Analyze customer groups.

Output

The algorithm groups customers into different clusters based on similar purchasing behavior and income levels.

Applications

- Customer Segmentation
- Targeted Marketing
- Business Strategy Planning
- Personalized Recommendations

Conclusion

K-Means Clustering was successfully implemented to segment customers into different groups. The resulting clusters help businesses understand customer behavior and improve marketing strategies.
